=head1 NAME

Web::Temma::Tokenizer - A tokenizer for L<Temma::Parser>

=head1 DESCRIPTION

The C<Web::Temma::Tokenizer> is a tokenizer for the L<Temma::Parser>
module in the Temma package <https://github.com/wakaba/temma>.  It
should not be used for any other purpose.
